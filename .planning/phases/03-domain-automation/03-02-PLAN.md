# Plan 03-02: Cloudflare Registrar Integration

## Objective
Integrate Cloudflare Registrar API for at-cost domain purchases.

## Tasks

1. **Cloudflare API Client**
   - API token authentication
   - Domain search/availability check
   - Domain registration with Cloudflare
   - DNS zone creation

2. **Domain Purchase Flow**
   - Search available domains
   - Check pricing
   - Purchase domain
   - Auto-create DNS zone

3. **Configuration**
   - Store Cloudflare API credentials
   - Handle account linking
