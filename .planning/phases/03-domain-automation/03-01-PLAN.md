# Plan 03-01: Domain Management Schema and UI

## Objective
Create database schema, CRUD API, and UI for managing domains with health tracking.

## Tasks

1. **Domain CRUD API**
   - GET /api/domains - List all domains for org
   - POST /api/domains - Add manual domain
   - PATCH /api/domains/[id] - Update domain settings
   - DELETE /api/domains/[id] - Remove domain
   - POST /api/domains/[id]/verify - Verify DNS configuration

2. **Domain Management UI**
   - Domain list with health status indicators
   - Add domain modal (manual entry)
   - Domain details panel with DNS status
   - Delete domain confirmation

3. **Health Monitoring**
   - DNS health check endpoint
   - SPF/DKIM/DMARC status indicators
   - Scheduled health checks (cron)
