# Plan 01-03: Authentication and Dashboard Layout

## Objective
Implement Supabase authentication flow and create the main dashboard layout with navigation.

## Tasks

### Task 1: Install Supabase Packages
```bash
npm install @supabase/supabase-js @supabase/ssr
```

### Task 2: Create Supabase Client Utilities
Create browser and server clients for Supabase.

### Task 3: Implement Auth Middleware
Create middleware to protect dashboard routes and redirect unauthenticated users.

### Task 4: Create Auth Pages
- Login page with email/password
- Register page with organization creation
- Password reset flow

### Task 5: Create Dashboard Layout
- Sidebar navigation with icons
- Header with user menu
- Main content area
- Mobile-responsive design

### Task 6: Create Navigation Structure
```
Dashboard
├── Overview (/)
├── Campaigns (/campaigns)
├── Leads (/leads)
├── Email Accounts (/accounts)
├── Domains (/domains)
├── Warmup (/warmup)
├── Inbox (/inbox)
├── Analytics (/analytics)
└── Settings (/settings)
```

### Task 7: Implement User Context
Create React context for current user and organization data.

## Verification
- Can register new user and create organization
- Can login/logout
- Dashboard protected routes work
- Sidebar navigation functional
- User menu with profile/logout

## Done When
- Auth flow complete (register → login → dashboard)
- Dashboard layout with working navigation
- Protected routes enforced
- User context available throughout app
