# Plan 01-01: Next.js Project Setup

## Objective
Initialize Next.js 14 project with TypeScript, Tailwind CSS, and shadcn/ui component library.

## Tasks

### Task 1: Initialize Next.js Project
```bash
npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --use-npm
```

### Task 2: Install and Configure shadcn/ui
```bash
npx shadcn@latest init
```
- Style: New York
- Base color: Slate
- CSS variables: Yes

Install core components:
```bash
npx shadcn@latest add button card input label form table dialog dropdown-menu avatar badge separator tabs toast sheet sidebar
```

Create `.env.example` and `.env.local` with:
```

SUPABASE_SERVICE_ROLE_KEY=
```
